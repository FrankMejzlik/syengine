◆ eEngineError

enum SYE::eEngineError
strong

All possible error types in engine.

1 - 99 -> Critical errors -> Will terminate as soon as possible. 100 - 499 -> Errors -> Will try to handle them. 500 - 999 -> Warnings -> Notify user and try to solve them. 1000+ -> Warnings to event that are supposed to happen e.g. Attaching duplicate Components, ...
