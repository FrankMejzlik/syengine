Engine concepts

Brief concepts

Game (Scene) consists of Entities. Entities are defined by their Primary Components, Entity without Components is nothing but useless data in memory. Primary Components are specific classes with specific functionalities and responsibilities and can have SubComponents

Example hierarchy

  • Entity
    • Transform
    • MeshRenderer
      • Mesh
      • Material
        • Texture
        • Shininess
        • Shader
      • Rigidbody
      • SphereCollider
        • Mesh
      • ScriptHandler
        • BallController (inherits from Script class)

There are implemented some shorthands for frequently used standard Entities like Camera, Block, Sphere etc. You can find Create{Something} methods in Scene class.
