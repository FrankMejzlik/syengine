Basic Scene composition

Game (Scene) consists of Entities.

Entities are defined by their Primary Components, Entity without Components is nothing but useless data in memory.

Primary Components are specific classes with specific functionalities and responsibilities. Some of them have SubComponents attached to them.

"Example hierarchy"

  • Entity
    • Transform
    • MeshRenderer
      • Mesh
      • Material
        • Texture
        • Shininess
        • Shader
      • Rigidbody
      • SphereCollider
        • Mesh
      • ScriptHandler
        • BallController (inherits from Script class)
